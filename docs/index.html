How It Works

  • Single-letter tiles: Can be used in any direction
  • Horizontal tiles: Stay together horizontally, split for vertical words
  • Vertical tiles: Stay together vertically, split for horizontal words
  • Group exclusion: Can't use multiple letters from the same split tile in one word
  • Minimum length: 3 letters per word
